# Detail Harvester MVP v5

Local web app for extracting and cataloging construction drawing details from PDF drawing sets.

## v5 changes

- Uses cursor-position hit testing instead of hidden handle elements.
- Move: click/drag inside a box.
- Resize: click/drag near any edge or corner.
- Right-click drag pans.
- Large gray workspace margin around the sheet so you can pan past the drawing edges.
- Box outlines stay visually consistent while zooming.
- Overlap merge still works.
