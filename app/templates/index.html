Review Controls

Wheel = zoom. Right-click drag anywhere in the gray workspace = pan. Click-drag empty sheet area = draw a new box. Drag inside a box = move. Drag near any box edge/corner = resize. Delete key removes selected box; matching detector boxes on later sheets in this upload are hidden automatically. If a dragged box overlaps one or more boxes by 50%+, the group is merged into the largest box on mouse-up.
